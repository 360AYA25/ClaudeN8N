# ClaudeN8N

Claude Code project for n8n workflow automation.

## Overview

This project provides tools and utilities for working with n8n workflows using Claude Code.

## Project Structure

```
ClaudeN8N/
├── docs/           # Documentation
├── templates/      # Workflow templates (JSON)
├── .mcp.json       # MCP server configuration
└── .env.example    # Environment variables
```

## Setup

1. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

2. Edit `.env` with your credentials

3. Configure MCP server in `.mcp.json`
